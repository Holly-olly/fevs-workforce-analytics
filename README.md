# fevs-workforce-analytics

Fpredicting employee engagement and performance confidence

## Status

Setup complete — analysis not yet started.

## Setup

```bash
pip install -r requirements.txt
cp .env.example .env
# Add your API keys to .env
```

## Project Structure

- `data/` — Raw data files
- `notebooks/` — Jupyter notebooks for analysis
- `outputs/` — Generated outputs and figures
- `claude.md` — Project context for Claude

## Getting Started

1. Add project context to `claude.md`
2. Add dependencies to `requirements.txt`
3. Copy `.env.example` to `.env` and fill in your API keys
4. Add your data files to `data/`
